* Git Commands
- $ git remote -v : To check the nickname of repository.
- $ git fetch origin : To fetch the latest updates of repository. 
- $ git checkout -b <branch> origin/main : To create branch from main branch.
- $ git satus : To check modification done in out files.
- $ git add <file> : To add files.
- $ git commit -m "<description message>" : commit your changes with neccesary message of your changes.
- $ git push origin batman-bike : push your changes.
- $ git checkout main : To switch main branch.
- $ git merge --no-ff origin/batman-bike : To merge chnges with main branch.
